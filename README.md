# Berkeley Public Data

Independent public-interest repository for Berkeley, California civic data, source documents, and interactive visualizations.

## Current visualizations

- [FY2027 Community Agency Allocations Sankey](./charts/fy2027-community-agency-sankey/)

## Scope

This repository collects, cleans, documents, and visualizes public civic data related to Berkeley, California, including budgets, policies, public records, council materials, and city services.

## Source materials

Source documents, public records, and third-party datasets remain under their original terms. They are included or linked for reference, attribution, and reproducibility.
